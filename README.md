## Featured Work

**[Network Traffic Monitor](https://github.com/kamer-heap/network-traffic-monitor)**
Real-time traffic dashboard built with Flask and Scapy — captures and visualizes live packet data across 20+ ports, with a simulation mode for offline demos.
`Python` `Flask` `Scapy`

**[Log Analyzer](https://github.com/kamer-heap/log-analyzer)**
CLI + web tool that parses server logs and handles 38+ real-world edge cases (malformed entries, timezone shifts, truncated lines) — tracks status codes, request volume, and traffic anomalies.
`Python` · [Live Demo](https://kamer-heap.github.io/log-analyzer/web_ui.html)

**[KadEdit](https://github.com/kamer-heap/KadEdit)**
Browser-based text editor inspired by VS Code's dark theme — implements undo/redo via a custom stack, recent-file queues, and dynamic buffer management.
`C++` `HTML` · [Live Demo](https://kamer-heap.github.io/KadEdit/)

**[Chrome Dino Game](https://github.com/kamer-heap/CHROME-DINO-GAME)**
Endless-runner clone with a custom collision detection engine and geometric bounding-box logic, built from scratch in C++.
`C++`

**[Minesweeper](https://github.com/kamer-heap/Minesweeper-game-)**
Console game with scaling difficulty, a custom flagging system, persistent high scores, and a recursive auto-reveal algorithm.
`C++` · [Live Demo](https://kamer-heap.github.io/Minesweeper-game-/)

**[Infix Engine](https://github.com/kamer-heap/infix-engine)**
Expression parser and evaluator with structural algorithm optimizations for parsing and validation.
`C++` · [Live Demo](https://kamer-heap.github.io/infix-engine/)
